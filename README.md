## Introduction
JouGan can download the file you request. I calculate and report:   
- Time taken to download the file (seconds)
- Download speed (MB/s)
- Time taken to save the file (seconds)
- Save speed (MB/s)
- Time taken to delete the file (seconds)
- Delete speed (MB/s)
